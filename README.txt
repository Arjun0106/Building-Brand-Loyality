This woocommerce extension allow merchants to reward their customers with loyalty points. 

== Description ==

**Points and Rewards for WooCommerce** plugin allow the merchant to reward their customers with the points on the signup, purchasing and other activities performed on the site.

The customers can use these points as a discount on purchase as well as get  the membership.
